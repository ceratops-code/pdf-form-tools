# pdf-form-tools

`pdf-form-tools` is a local Python package for filling layout-sensitive scanned PDF forms with deterministic placement and visual verification.

The package uses a standard `src` layout:

- repo root: packaging, git metadata, docs
- `src/pdf_form_tools`: importable Python package
- installed command: `pdf-form-tools`

This repo is the editable source of truth. The installed wheel in `site-packages` is only a built runtime copy.
