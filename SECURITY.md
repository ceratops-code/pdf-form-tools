# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 2.x     | Yes       |
| < 2.0   | No        |

## Reporting a Vulnerability

Do not open a public issue for a vulnerability that could expose user documents or private data.

For sensitive reports, email `roman.ost85@gmail.com` with:

- affected version
- impact
- reproduction details
- any workaround you found

For non-sensitive hardening issues, a normal GitHub issue is fine.
